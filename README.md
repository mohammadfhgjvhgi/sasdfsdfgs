## 📋 الوصف

نظام كراج ذكي متكامل يتيح:
- **المراقبة الفورية** لعدد السيارات و حالة المستشعرات
- **التحكم عن بُعد** بالبوابات والإضاءة
- **نظام الإنذار** للحريق والغاز والاهتزاز
- **حجز المواقف** إلكترونياً
- **تقارير وإحصائيات** شاملة

---

## 🔧 القطع المطلوبة

### المتحكمات
| القطعة | الكمية | الوصف |
|--------|--------|-------|
| Arduino Mega 2560 | 1 | المتحكم الرئيسي |
| ESP8266 | 1 | وحدة WiFi (مدمجة أو خارجية) |

### المحركات
| القطعة | المنفذ | الوصف |
|--------|--------|-------|
| Servo Motor | D9 | بوابة الدخول |
| Servo Motor | D10 | بوابة الخروج |
| Servo Motor | D53 | باب غرفة الأمان |

### المستشعرات
| القطعة | المنفذ | الوصف |
|--------|--------|-------|
| IR Sensor | D27, D28 | كشف السيارات |
| DHT11 | D26 | الحرارة والرطوبة |
| MQ-2 Gas | A0 | كشف الغاز |
| Flame Sensor | D22 | كشف الحريق |
| Vibration | D23 | كشف الاهتزاز |

### المكونات الأخرى
| القطعة | المنفذ | الوصف |
|--------|--------|-------|
| LCD 16x2 I2C | SDA/SCL | شاشة العرض |
| RTC DS3231 | SDA/SCL | الساعة |
| Buzzer | D24 | جرس الإنذار |
| Relay | D25 | تحكم الإضاءة |
| RFID MFRC522 | D50-D53, D8 | فتح الباب |

## 📱 مميزات التطبيق

### لوحة التحكم
- إحصائيات فورية (عدد السيارات، الحرارة، الرطوبة)
- رسوم بيانية حية
- سجل الأحداث

### التحكم بالبوابات
- بوابة الدخول (D9)
- بوابة الخروج (D10)
- باب غرفة الأمان (D53)

### المستشعرات
- IR للدخول والخروج
- DHT11 للبيئة
- MQ-2 للغاز
- حساس اللهب
- حساس الاهتزاز

### نظام الحجز
- 20 موقف
- حجز إلكتروني
- حساب الرسوم

### التقارير
- يومي / أسبوعي / شهري
- تصدير PDF / Excel
- رسوم بيانية متقدمة

## 🚀 طريقة التشغيل

### 1. تشغيل التطبيق
```bash
# افتح الملف في المتصفح
index.html
```

### 2. ربط الأردوينو
1. ارفع كود ESP8266 على وحدة WiFi
2. ارفع كود Mega 2560 على المتحكم
3. اضبط مفاتيح DIP للتشغيل
